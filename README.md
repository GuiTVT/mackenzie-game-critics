# Mackenzie Game Critics

Projeto acadêmico desenvolvido na Universidade Presbiteriana Mackenzie.

O objetivo do projeto é analisar dados relacionados aos jogos indicados à categoria **Game of the Year** da premiação **The Game Awards**, entre os anos de 2014 e 2025.

- Bruna Graciano
- Guilherme Marques Gomes dos Santos

## Objetivo

Criar uma plataforma de análise e visualização de dados sobre jogos indicados ao Game of the Year, utilizando ferramentas de análise de dados e visualização.

## Tecnologias utilizadas

- Python (para análise de dados)
- Power BI (visualização de dados)
- HTML5 e CSS3 (estrutura do site)
- GitHub (versionamento do projeto)

## Dataset

A base de dados utilizada contém informações sobre os jogos indicados ao prêmio Game of the Year entre 2014 e 2025, incluindo:

- Nome do jogo
- Ano da indicação
- Diretor
- Quantidade de prêmios
- Vendas aproximadas
- Plataforma
- Gênero
